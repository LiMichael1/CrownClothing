## Requirements

- npm
- Firebase

## Installation Guide

> git clone https://github.com/LiMichael1/CrownClothing
> cd CrownClothing
> npm install

Edit the .env file to include your Firebase Settings

## Run the Project

> npm start

Navigate browser to localhost:3000/
